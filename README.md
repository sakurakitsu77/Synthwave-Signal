# Synthwave Signal

A cozy, synthwave-themed mobile game about restoring a lost radio track across space.

## Files

- `index.html` — app shell and overlays
- `styles.css` — full-screen synthwave UI
- `audio.js` — procedural Web Audio synth engine
- `game.js` — canvas gameplay and controls

## Gameplay

You drift left and right in the lower rail of the screen and collect glowing notes.
Every note restores part of the song. The goal is not survival. The goal is to finish the track.

## Vibe

- retro synthwave colors
- calm space visuals
- soft glass UI
- music-first progression
- no harsh game over
